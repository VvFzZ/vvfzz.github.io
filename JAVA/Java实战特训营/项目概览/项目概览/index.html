技术需求分类：

  • 业务类型技术需求
  • 基础设施类技术需求

业务需求

系统集成，异构系统数据继承
即时通讯，
场景搜索，
智能回复，
系统权限（安全），建立多角色账户体系

技术需求

  • 数据处理，海量数据存储，高效查询，且保证一致性
  • 系统扩展，扩展变化时不可影响现有业务
  • 系统解耦，降低不同业务边界间的依赖
  • 系统可用，流量管理和服务容错
  • 系统性能，监控并优化，对系统状态实施观察，根据获取的性能数据和指定的性能指标，通过性能优化策略合技术规避系统瓶颈
  • 系统安全，网络攻击处理
