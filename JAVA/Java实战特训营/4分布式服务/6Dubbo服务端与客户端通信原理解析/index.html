Dubbo网络通信架构

  • Protocol层 业务层
  • Exchange层 信息交换层，用来封装请求-响应模式（心跳处理）
  • Transport层 网络传输层，抽象Netty等框架作为统一的接口
  • Serialize层 序列化层，主要完成数据的序列化和反序列化过
